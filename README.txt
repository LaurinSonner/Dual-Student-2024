Dies ist eine Webapplikation um Kundendaten aus einer SQLite Database zu lesen und anzuzeigen. Das Python Skript erstellt aus der .csv eine Datenbank und speichert die Daten darin.
Um die Webapplikation zu benutzen muss der Projektordner in einer IDE aufgemacht werden (Visual Studio Code) und in der Konsole der Befehl der node server.js eingegeben werden.
Dadurch wird der Webserver auf Local Host 3000 gestartet und man kann in einem Webbrowser unter der Adresse http://localhost:3000/ die Webapplikation nutzen.
